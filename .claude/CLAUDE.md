## 한국사 연표 작성 규칙

한국사 연표는 매우 중요하므로 정확한 연도와 사건 순서를 최우선으로 검증해서 작성한다.

연표는 고구려, 백제, 신라처럼 국가별로 나누지 않는다. 모든 국가와 사건을 하나의 통합 연표로 모아 연도별, 사건별 순서대로 정렬한다. 예를 들어 소수림왕의 율령 반포와 근초고왕의 사망처럼 서로 다른 국가의 사건도 같은 흐름 안에서 비교할 수 있게 작성한다.

한능검용 삼국 연표에서는 건국 전승 연도(신라 기원전 57, 고구려 기원전 37, 백제 기원전 18)를 핵심 흐름으로 앞세우지 않는다. 한능검은 삼국의 중앙집권 고대 국가화, 체제 정비, 왕별 업적, 전성기, 사료 키워드를 더 중요하게 묻는다. 따라서 고구려 태조왕·고국천왕·소수림왕, 백제 고이왕·근초고왕, 신라 내물왕·지증왕·법흥왕·진흥왕의 흐름을 우선한다.

## Git Push 방법

이 프로젝트는 SSH 인증으로 push한다. remote URL이 SSH로 설정되어 있어야 한다.

```
git remote set-url origin git@github.com:jammy0903/aingan-study.git
git push
```

HTTPS로 push하면 인증 실패가 난다. 항상 SSH remote URL을 사용할 것.
