# 계획 — analysis.html 뱃지 → 레슨 페이지 딥링크

## 목표

`hankuksa/analysis.html`의 최빈출/중빈출/트렌드/선지 전용 뱃지를 클릭하면, **해당 단원 레슨 페이지가 열리면서 자동으로 그 개념만 검색·필터링**된 상태로 보이게 한다.

예: "계백" 뱃지 클릭 → `/hankuksa/02/?q=계백` → 고대 레슨 페이지 로드 시 검색창에 "계백" 자동 입력 + 해당 카드만 표시.

## 단원 매핑

| 라벨 | 경로 |
|---|---|
| 선사 | `/hankuksa/01/` |
| 고대 | `/hankuksa/02/` |
| 고려 | `/hankuksa/03/` |
| 조선 | `/hankuksa/04/` |
| 근대 | `/hankuksa/05/` |
| 일제 | `/hankuksa/06/` |
| 현대 | `/hankuksa/07/` |

## 구현 단계

### 1. 레슨 페이지 7개에 쿼리 파라미터 수신 코드 추가

각 `hankuksa/0N/index.html` 의 `applyFilters()` 선언 직후에 아래 스니펫 삽입.

```js
// URL ?q=xxx 가 있으면 검색창 자동 채우기 + 필터 실행
(function initFromQuery(){
  const params = new URLSearchParams(location.search);
  const q = params.get('q');
  if (q) {
    const input = document.querySelector('.search');
    if (input) {
      input.value = q;
      applyFilters();
      input.focus();
    }
  }
})();
```

대상 파일: `01~07/index.html` (총 7개).

### 2. `analysis.html` 뱃지 전수 변환

현재 `<span class="hit">…</span>` → `<a class="hit" href="./0N/?q=라벨">…</a>`.

변환 대상 섹션(총 ~50개):
- 1순위 TOP 테이블 8행 → 각 행의 `.label` td 를 링크로
- 2순위 중빈출 뱃지 18개
- 3순위 최근 트렌드 뱃지 13개
- 선지 전용 뱃지 10개

자동화 스크립트 권장:
- 각 뱃지의 라벨 텍스트와 단원 표기("고려", "조선" 등)를 파싱
- 단원명 → 경로 매핑 적용
- `<a href="./0N/?q={라벨}">` 로 일괄 치환

수동 수정 시간: 약 30분. 스크립트 작성 포함 시 약 20분.

### 3. 스타일 보정

`<a>` 태그가 되면 브라우저 기본 밑줄·보라색 방문 링크 스타일이 튐. CSS에 추가:

```css
a.hit{text-decoration:none}
a.hit:visited{color:var(--ink)}
a.hit:hover{background:#f8f0e0;border-color:#d4a76a}
```

### 4. 검증

- 계백 뱃지 클릭 → 고대 페이지 열리고 "계백" 검색된 상태 ✓
- 정우회 선언 클릭 → 일제 페이지에서 "정우회 선언" 필터링 ✓
- 브라우저 뒤로가기로 analysis.html 복귀 정상 ✓
- 모바일 터치도 정상 작동 ✓

## 예외 처리

일부 뱃지는 단원 정보가 애매함:
- `2·8 독립선언` — 일제 (3·1운동 배경)
- `김기수·영선사·메가타` — 근대 (사절단, 복합 라벨)
- `영선사(1881)` — 근대, 괄호 연도 포함

복합 라벨 처리:
- `김기수·영선사·메가타` → `?q=영선사` 처럼 대표 키워드 하나만 사용
- 괄호 연도는 제거: `영선사(1881)` → `?q=영선사`

## 대안 (간단 버전)

검색 필터링까지 필요 없고 단원 인덱스 페이지로만 이동해도 충분하다고 판단될 경우:
- 1단계 생략
- 2단계에서 `href="./0N/?q=…"` 대신 `href="./0N/"` 만 사용
- 구현 시간 5분
