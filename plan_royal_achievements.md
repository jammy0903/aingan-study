# 한국사 왕 업적 연도표 구축 계획

## 목표

한국사 학습자가 왕별 업적을 국가 구분 없이 연도순으로 빠르게 훑어볼 수 있는 별도 화면을 만든다.

현재 `/hankuksa/timeline/`은 사건 중심 연도표이고, 각 단원 페이지의 `LESSONS` 데이터도 HTML 안에 직접 포함되어 있다. 따라서 왕 업적 전용 데이터를 새로 만들고, 이를 정적 JSON으로 관리한다.

## 결론

DB가 아니라 JSON으로 구축한다.

이 프로젝트는 정적 HTML 중심 구조라 DB를 붙이면 서버, API, 쿼리 계층이 추가되어 복잡도가 커진다. 왕 업적 데이터는 수백 건 규모면 충분하므로 JSON 파일을 fetch해서 브라우저에서 필터링하는 방식이 더 적합하다.

## 파일 구조

```text
hankuksa/
  data/
    royal-achievements.json
  kings/
    index.html
```

추가 연결:

```text
hankuksa/
  index.html
  timeline/
    index.html
```

`hankuksa/index.html`에는 "왕 업적 연도표" 진입 링크를 추가한다.  
`hankuksa/timeline/index.html`에는 "왕 업적" 바로가기 버튼을 추가한다.

## JSON 스키마

```json
{
  "id": "goguryeo-sosurim-372",
  "year": "372",
  "sort": 372,
  "dynasty": "고구려",
  "period": "삼국·남북국",
  "king": "소수림왕",
  "title": "태학 설립·불교 수용",
  "achievement": "전진에서 불교를 수용하고 태학을 설립해 국가 체제를 정비했다.",
  "category": "제도",
  "keywords": ["태학", "불교", "중앙집권"],
  "href": "../02/?q=소수림왕"
}
```

필드 기준:

| 필드 | 의미 |
| --- | --- |
| `id` | 고유 식별자. `국가-왕-연도` 형태 |
| `year` | 화면에 표시할 연도 |
| `sort` | 정렬용 숫자 |
| `dynasty` | 고구려, 백제, 신라, 통일신라, 발해, 고려, 조선 등 |
| `period` | 초기국가, 삼국·남북국, 후삼국, 고려, 조선 등 큰 시대 |
| `king` | 왕 이름 |
| `title` | 업적 카드 제목 |
| `achievement` | 한 줄 설명 |
| `category` | 정치, 제도, 영토, 외교, 문화, 경제, 군사 등 |
| `keywords` | 검색과 복습용 키워드 |
| `href` | 관련 단원 검색 링크 |

## 1차 데이터 범위

처음부터 모든 왕을 넣기보다 한능검 심화에서 반복 출제되는 왕을 우선 구축한다.

### 고구려

- 태조왕: 옥저 복속, 계루부 고씨 왕위 세습
- 고국천왕: 부자 상속 강화, 진대법
- 미천왕: 서안평 점령, 낙랑군 축출, 대방군 점령
- 소수림왕: 불교 수용, 태학 설립, 율령 반포
- 광개토대왕: 정복 활동, 신라 지원, 영락
- 장수왕: 평양 천도, 남진 정책, 백제 한성 공격

### 백제

- 고이왕: 관등제, 관복제, 한강 유역 장악
- 근초고왕: 마한 정복, 부자 상속제, 서기 편찬, 대외 진출
- 침류왕: 불교 수용
- 무령왕: 22담로, 지방 통제, 무령왕릉
- 성왕: 사비 천도, 남부여, 일본 불교 전파, 관산성 전투
- 의자왕: 대야성 함락, 백제 멸망

### 신라·통일신라

- 내물왕: 김씨 왕위 세습, 마립간
- 지증왕: 국호 신라, 왕호 사용, 우산국 복속
- 법흥왕: 율령 반포, 불교 공인, 금관가야 병합
- 진흥왕: 한강 유역 장악, 대가야 병합, 순수비
- 선덕여왕: 첨성대, 불교 문화
- 문무왕: 삼국 통일 완성
- 신문왕: 김흠돌의 난 진압, 국학, 9주 5소경, 관료전, 녹읍 폐지
- 성덕왕: 정전 지급
- 경덕왕: 녹읍 부활
- 원성왕: 독서삼품과

### 발해

- 대조영: 발해 건국
- 무왕: 당 등주 공격
- 문왕: 3성 6부, 상경 용천부, 당 문물 수용
- 선왕: 해동성국, 5경 15부 62주

### 고려

- 태조 왕건: 고려 건국, 후삼국 통일, 북진 정책, 훈요 10조
- 광종: 노비안검법, 과거제, 칭제건원
- 경종: 전시과
- 성종: 최승로 시무 28조, 12목, 유교 정치
- 공민왕: 반원 개혁, 쌍성총관부 수복, 정동행성 이문소 폐지

### 조선

- 태조: 조선 건국
- 태종: 왕권 강화, 6조 직계제, 사병 혁파
- 세종: 훈민정음, 집현전, 과학 기술, 4군 6진
- 세조: 계유정난, 직전법, 6조 직계제 재시행
- 성종: 경국대전, 홍문관, 관수관급제
- 연산군: 무오사화, 갑자사화
- 중종: 조광조 개혁, 기묘사화
- 선조: 임진왜란, 훈련도감
- 광해군: 대동법, 중립 외교
- 인조: 인조반정, 정묘호란, 병자호란
- 숙종: 환국 정치, 상평통보 유통 확대
- 영조: 탕평책, 균역법
- 정조: 규장각, 장용영, 신해통공, 수원 화성
- 순조: 세도 정치, 신유박해, 홍경래의 난
- 고종: 대한제국, 광무개혁

## 화면 요구사항

경로: `/hankuksa/kings/`

기본 화면:

- 상단 제목: `왕 업적 연도표`
- 검색 입력: 왕 이름, 업적, 키워드 검색
- 필터 칩: `전체`, `고구려`, `백제`, `신라`, `통일신라`, `발해`, `고려`, `조선`
- 정렬: `sort` 오름차순
- 카드: 연도, 국가, 왕, 업적명, 설명, 키워드, 관련 레슨 링크

카드 예시:

```text
372
고구려 · 소수림왕 · 제도
태학 설립·불교 수용
전진에서 불교를 수용하고 태학을 설립해 국가 체제를 정비했다.
태학 / 불교 / 중앙집권
```

## 구현 단계

### 1단계: 데이터 파일 생성

- `hankuksa/data/royal-achievements.json` 생성
- 1차 왕 업적 데이터 입력
- 기존 `timeline` 데이터와 중복되는 사건은 연도와 설명을 맞춘다

### 2단계: 왕 업적 페이지 생성

- `hankuksa/kings/index.html` 생성
- JSON fetch
- 연도순 렌더링
- 검색/필터 구현
- 모바일에서도 가로 스크롤 없이 보이게 구성

### 3단계: 기존 페이지 연결

- `hankuksa/index.html`에 "왕 업적 연도표" 항목 추가
- `hankuksa/timeline/index.html`에 "왕 업적" 링크 추가

### 4단계: 검증

- JSON 파싱 오류 없음
- `sort` 기준 정렬 확인
- 필터별 결과 수 확인
- 검색어 예시 확인: `소수림왕`, `율령`, `규장각`, `탕평`, `쌍성총관부`
- 관련 단원 링크 이동 확인

## 완료 기준

- `/hankuksa/kings/`에서 왕 업적이 연도순으로 표시된다
- 국가 필터가 정상 동작한다
- 검색이 왕 이름, 업적명, 키워드에 모두 걸린다
- 각 카드에서 관련 단원으로 이동할 수 있다
- 왕 업적 데이터가 HTML이 아니라 JSON으로 분리되어 git diff가 읽기 쉽다

## 이후 확장

필요해지면 다음 기능을 추가한다.

- 왕별 묶어보기
- 국가별 왕 순서 보기

- 빈출도 표시
- 기출 선지 연결
- 즐겨찾기/복습 체크
