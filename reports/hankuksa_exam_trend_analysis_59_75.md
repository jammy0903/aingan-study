# 한능검 심화 59~75회 출제 경향 분석

분석 범위: 한국사능력검정시험 심화 59회~75회 해설집 PDF 17개

분석 목적: 내용을 단순 요약하는 것이 아니라, 문제가 어떤 방식으로 출제되는지, 어디서 꼬는지, 어떻게 공부해야 하는지를 정리한다.

주의: 해설집 PDF는 2단 편집이라 텍스트 추출 시 문항 순서가 일부 섞인다. 따라서 이 문서는 문항별 정답 데이터베이스가 아니라, 17회분 해설 텍스트에서 반복되는 출제 방식과 함정 구조를 분석한 것이다.

## 1. 결론

한능검 심화는 "많이 외운 사람"보다 "단서를 보고 시대와 주체를 바로 좁히는 사람"에게 유리하다.

문제의 기본 구조는 대체로 다음 순서다.

1. 지문, 사진, 지도, 검색창, 대화, 사료에서 결정적 단서 1~3개를 준다.
2. 그 단서로 시대, 국가, 왕, 단체, 지역을 먼저 특정해야 한다.
3. 선지는 비슷한 시기나 비슷한 기능의 사건으로 섞어 낸다.
4. 정답은 지문 전체를 번역해서 찾는 것이 아니라, 핵심 단서와 연표 위치를 맞혀서 고른다.

즉 공부는 "설명문 암기"보다 아래 형태가 더 효율적이다.

- 키워드 -> 시대/국가/왕/단체
- 사건 A -> 사건 B 사이에 들어갈 사건
- 왕/정부별 업적 비교
- 문화재/지역/단체별 한 줄 단서
- 오답 선지가 왜 틀렸는지 분류

## 2. 출제 방식 집계

해설 텍스트에서 반복된 문항 표현과 키워드를 집계하면 다음 경향이 강하다.

| 유형 | 17회분 반복 횟수 | 의미 |
|---|---:|---|
| 자료/사료 판독형 | 317 | 사료, 대화, 신문, 검색창, 사진에서 키워드를 찾는 유형이 매우 많다. |
| 지역/지도/문화재형 | 303 | 문화유산, 유적, 지역 답사, 지도 문제는 매회 반복된다. |
| 사이 시기/연표형 | 167 | "(가), (나) 사이", "이후에 있었던 사실"이 계속 나온다. |
| 왕/인물 식별형 | 118 | 왕의 재위 시기, 인물 업적, 정부별 정책을 묻는다. |
| 국가/나라 식별형 | 53 | 고대 국가, 삼국, 발해, 고려 문화 등에서 주로 나온다. |
| 단체/운동/정책형 | 895 | 근현대와 일제 강점기에서 특히 강하다. |

이 숫자는 "정확한 문제 수"가 아니라 추출 텍스트 기준 반복 신호다. 그래도 방향은 분명하다. 한능검은 키워드 판독과 연표 위치 판정이 핵심이다.

## 3. 시대별 출제 강도

키워드 빈도 기준으로는 고대~조선~근현대가 고르게 나오지만, 출제 방식은 시대마다 다르다.

| 시대 | 출제 성격 |
|---|---|
| 선사/초기 국가 | 생활 도구, 토기, 무덤, 제천 행사, 여러 나라 특징을 빠르게 식별 |
| 삼국/가야/발해 | 왕 업적, 국가별 문화재, 전쟁과 외교, 통일 과정, 발해 제도 |
| 고려 | 왕별 제도, 대외 항쟁, 무신 정권, 원 간섭기, 공민왕 개혁 |
| 조선 전기 | 왕별 제도, 유교 통치 기구, 사화, 임진왜란 전후 |
| 조선 후기 | 붕당, 탕평, 수취 제도, 상품 화폐 경제, 실학, 세도 정치 |
| 근대 | 조약과 개혁의 순서, 위정척사/개화/동학/의병/애국계몽 구분 |
| 일제 강점기 | 1910/1920/1930~40년대 정책, 단체, 무장 투쟁, 사회 운동 |
| 현대 | 정부별 정책, 민주화 운동 순서, 통일 정책, 경제 정책 |

## 4. 가장 중요한 출제 구조

### 4.1 단서 1개로 국가/시대 맞히기

지문에 나오는 한 단어가 거의 답을 결정한다.

예시 단서:

| 단서 | 바로 연결할 것 |
|---|---|
| 사유 재산, 계급, 고인돌, 비파형 동검 | 청동기 |
| 주먹도끼, 찍개, 동굴, 막집 | 구석기 |
| 가락바퀴, 빗살무늬 토기 | 신석기 |
| 사출도, 영고 | 부여 |
| 민며느리제, 가족 공동 무덤 | 옥저 |
| 책화, 무천 | 동예 |
| 신지, 읍차, 제정 분리 | 삼한 |
| 5소경 | 통일신라 |
| 법화원 | 장보고 |
| 인안, 대흥 | 발해 무왕/문왕 |
| 주자감 | 발해 |
| 12목 | 고려 성종 |
| 광덕, 준풍 | 고려 광종 |
| 전민변정도감, 쌍성총관부 | 공민왕 |
| 은대, 승지 | 승정원 |
| 광성보, 어재연 | 신미양요 |
| 식량 배급, 강제 동원 | 1930년대 이후 전시 동원 체제 |

공부법: 긴 설명을 외우기보다 "단서 -> 답" 카드로 만들어야 한다.

### 4.2 연표 사이 시기 문제

출제자가 가장 자주 꼬는 방식이다.

대표 표현:

- "(가), (나) 사이의 시기에 있었던 사실"
- "다음 상황 이후에 있었던 사실"
- "다음 사건이 일어난 순서"
- "조약 체결 사이의 시기"
- "헌법이 제정된 시기 사이"

이 유형은 사건 자체를 알아도 앞뒤 순서를 모르면 틀린다.

공부법:

1. 사건을 낱개로 외우지 말고 5~7개짜리 흐름으로 외운다.
2. "정확한 연도"보다 "앞뒤 위치"를 먼저 잡는다.
3. 오답 선지가 어느 쪽으로 벗어났는지 표시한다.

예시 흐름:

- 백제 멸망 -> 백제 부흥운동 -> 백강 전투
- 매소성 -> 기벌포 -> 신라의 삼국 통일
- 묘청의 서경 천도 운동 -> 무신정변 -> 망이·망소이의 난 -> 최충헌 집권
- 강화도 조약 -> 통리기무아문 -> 임오군란 -> 갑신정변
- 동학 농민 운동 -> 갑오개혁 -> 을미사변 -> 아관파천 -> 독립협회
- 3.1 운동 -> 대한민국 임시정부 -> 문화 통치 -> 신간회
- 4.19 혁명 -> 5.16 군사정변 -> 유신 -> 부마 항쟁/10.26 -> 5.18 -> 6월 민주항쟁

### 4.3 왕 업적 섞기

해설 텍스트에서 왕 이름 반복이 많다. 특히 다음 왕들이 자주 섞인다.

| 구간 | 자주 꼬는 왕 |
|---|---|
| 삼국 | 소수림왕, 근초고왕, 장수왕, 법흥왕, 진흥왕 |
| 통일신라 | 신문왕, 원성왕, 흥덕왕, 진성여왕 |
| 고려 | 광종, 성종, 공민왕 |
| 조선 | 세종, 세조, 성종, 영조, 정조, 순조 |

출제자는 보통 같은 계열의 업적을 서로 바꿔 놓는다.

대표 함정:

- 고구려 불교 수용/태학/율령 = 소수림왕
- 신라 불교 공인/이차돈/율령/건원 = 법흥왕
- 국사 편찬/한강 장악/대가야 병합/순수비 = 진흥왕
- 관료전 지급, 녹읍 폐지, 국학, 9주 5소경 = 신문왕
- 독서삼품과 = 원성왕
- 과거제 = 고려 광종
- 12목 파견, 최승로 시무 28조 수용 = 고려 성종
- 전민변정도감, 쌍성총관부 수복 = 공민왕
- 균역법 = 영조
- 규장각, 초계문신제, 수원 화성, 장용영 = 정조
- 신유박해, 세도 정치 시작 = 순조

공부법: 왕 이름을 단독 암기하지 말고 "직전 왕/직후 왕과 차이"로 외운다.

## 5. 자주 꼬는 지점

### 5.1 고대: 국가와 왕을 바꿔 치기

고대사는 한능검이 특히 "비슷한 키워드"를 다른 국가로 바꿔 낸다.

꼭 분리해야 할 것:

| 헷갈리는 것 | 정리 |
|---|---|
| 금관가야 vs 대가야 | 금관가야는 법흥왕 때 병합, 대가야는 진흥왕 때 병합 |
| 백제 성왕 vs 신라 진흥왕 | 성왕은 사비 천도/남부여/관산성 전사, 진흥왕은 한강 장악/순수비 |
| 소수림왕 vs 법흥왕 | 둘 다 율령·불교 축이지만 고구려는 소수림왕, 신라는 법흥왕 |
| 신문왕 vs 신라 하대 | 신문왕은 중대 왕권 강화, 김헌창의 난은 하대 |
| 독서삼품과 vs 과거제 | 독서삼품과는 원성왕, 과거제는 고려 광종 |
| 장보고 청해진 vs 장보고의 난 | 청해진 설치는 해상 무역, 장보고의 난은 중앙 진골 귀족과 충돌 |

신라 하대는 특히 이렇게 묶어야 한다.

혜공왕 이후 왕위 다툼 -> 원성왕 독서삼품과 -> 김헌창의 난(822) -> 장보고 청해진/장보고의 난(846) -> 원종·애노의 난(889) -> 최치원 시무 10여 조 -> 적고적의 난(896) -> 호족/선종/풍수지리설 -> 후삼국

핵심 함정:

- 김흠돌의 난은 신문왕, 김헌창의 난은 신라 하대
- 최치원 시무 10여 조와 최승로 시무 28조를 섞는다
- 장보고는 청해진/법화원/해상 무역으로도 나오고, 신라 하대 지방 세력으로도 나온다

### 5.2 고려: 왕별 제도와 대외 항쟁 순서

고려는 "왕별 업적"과 "대외 관계"를 섞는다.

필수 흐름:

- 태조 왕건: 후삼국 통일, 호족 포섭, 훈요 10조
- 광종: 노비안검법, 과거제, 광덕/준풍
- 경종: 전시과
- 성종: 최승로 시무 28조, 12목, 유교 정치
- 현종: 거란 침입, 초조대장경
- 문종: 최충, 사학 12도
- 숙종: 별무반
- 예종: 윤관, 동북 9성
- 인종: 이자겸의 난, 묘청의 서경 천도 운동
- 무신정변 이후: 정중부 -> 최충헌 -> 최우 -> 삼별초
- 공민왕: 반원 개혁, 쌍성총관부 수복, 전민변정도감

대표 함정:

- 12목은 성종, 과거제는 광종
- 동북 9성은 예종 때 윤관
- 전민변정 사업은 공민왕
- 삼별초는 몽골과 연결
- 왜구 토벌, 화통도감, 위화도 회군은 고려 말 흐름으로 묶기

### 5.3 조선: 왕/붕당/전쟁/경제를 섞기

조선은 왕 이름만 묻는 것처럼 보여도, 실제로는 제도와 시기 구분을 본다.

필수 비교:

| 왕/시기 | 키워드 |
|---|---|
| 태종 | 사병 혁파, 6조 직계제 |
| 세종 | 집현전, 훈민정음, 의정부 서사제, 4군 6진 |
| 세조 | 계유정난, 직전법, 6조 직계제 재시행 |
| 성종 | 경국대전, 홍문관, 관수관급제 |
| 연산군 | 무오사화, 갑자사화 |
| 중종 | 조광조, 현량과, 기묘사화 |
| 선조 | 임진왜란, 훈련도감 |
| 광해군 | 중립 외교, 대동법 |
| 인조 | 인조반정, 정묘호란, 병자호란 |
| 숙종 | 환국, 상평통보 |
| 영조 | 탕평책, 균역법, 이인좌의 난 진압 |
| 정조 | 규장각, 초계문신제, 신해통공, 장용영, 수원 화성 |
| 순조 | 세도 정치, 신유박해, 홍경래의 난 |

대표 함정:

- 영조와 정조 개혁을 섞는다
- 정조의 장용영과 순조의 세도 정치를 섞는다
- 속대전/대전통편/대전회통 계열을 왕별로 헷갈리게 낸다
- 임진왜란 이후와 병자호란 이후 사회 변화를 섞는다

### 5.4 근대: 사건 순서가 거의 전부

근대사는 사건을 개별 설명으로 외우면 잘 틀린다. 순서를 통째로 외워야 한다.

핵심 순서:

흥선대원군 통상 수교 거부 -> 강화도 조약 -> 개화 정책/통리기무아문 -> 임오군란 -> 갑신정변 -> 동학 농민 운동 -> 갑오개혁 -> 을미사변 -> 아관파천 -> 독립협회 -> 대한제국/광무개혁 -> 을사늑약 -> 정미7조약/군대 해산 -> 국권 피탈

대표 함정:

- 통리기무아문은 강화도 조약 이후, 임오군란 이전
- 갑신정변 결과와 동학 농민 운동 결과를 섞는다
- 을미사변, 아관파천, 독립협회, 대한제국 선포 순서를 묻는다
- 의병과 애국계몽운동을 같은 "항일"로 묶어 헷갈리게 낸다

### 5.5 일제 강점기: 1910/1920/1930년대 구분

일제 강점기는 연도대만 잡아도 선지 절반이 지워진다.

| 시기 | 핵심 키워드 |
|---|---|
| 1910년대 | 무단 통치, 헌병 경찰, 조선 태형령, 토지 조사 사업, 회사령 |
| 1920년대 | 문화 통치, 산미 증식 계획, 물산 장려 운동, 민립 대학 설립 운동, 신간회, 광주 학생 항일 운동 |
| 1930~40년대 | 민족 말살 통치, 국가 총동원법, 창씨개명, 징병제, 한국광복군 |

대표 함정:

- 회사령은 1910년대
- 산미 증식 계획은 1920년대
- 국가 총동원법은 1938년 이후
- 신간회는 1927년, 광주 학생 항일 운동 지원과 연결
- 의열단과 한인애국단을 섞는다
- 한국독립군, 조선혁명군, 한국광복군을 전투/지역/인물로 구분해야 한다

### 5.6 현대: 정부별 정책과 민주화 운동 순서

현대사는 정권별 키워드와 민주화 운동 순서를 섞는다.

필수 순서:

광복 -> 모스크바 3국 외상 회의 -> 좌우 합작 운동 -> 대한민국 정부 수립 -> 반민특위/농지개혁 -> 6.25 전쟁 -> 4.19 혁명 -> 5.16 군사정변 -> 유신 -> 부마 민주 항쟁/10.26 -> 5.18 민주화 운동 -> 6월 민주 항쟁 -> 6.29 선언 -> 노태우 정부 -> 김영삼 정부 -> 김대중 정부

대표 함정:

- 4.19와 5.16 순서
- 유신 이후 부마/10.26, 그 뒤 5.18
- 6월 민주 항쟁과 6.29 선언 연결
- 금융실명제는 김영삼
- 남북 유엔 동시 가입은 노태우
- 6.15 남북 공동 선언은 김대중

## 6. 공부법

### 6.1 1단계: 단서 카드 만들기

한능검 심화는 단서가 반복된다. 다음 형식으로 카드화한다.

| 앞면 | 뒷면 |
|---|---|
| 법화원 | 장보고, 청해진, 통일신라 하대 |
| 5소경 | 통일신라 |
| 광덕·준풍 | 고려 광종 |
| 12목 | 고려 성종 |
| 은대·승지 | 승정원 |
| 광성보·어재연 | 신미양요 |
| 회사령 | 1910년대 무단 통치 |
| 산미 증식 계획 | 1920년대 문화 통치 |
| 국가 총동원법 | 1938년 이후 전시 동원 |

카드는 길게 쓰지 않는다. 핵심 단서와 답만 연결한다.

### 6.2 2단계: 통합 연표로 사이 시기 훈련

연표는 국가별로 나누면 안 된다. 한능검은 같은 시기의 다른 국가 사건을 섞어 낸다.

예시:

- 근초고왕 평양성 공격 -> 소수림왕 체제 정비
- 성왕 관산성 전사 -> 진흥왕 한강 장악
- 백제 멸망 -> 백제 부흥운동 -> 백강 전투
- 고구려 멸망 -> 고구려 부흥운동 -> 나당 전쟁
- 신라 하대 반란 -> 후삼국 형성 -> 고려 건국

공부할 때는 사건마다 "직전 사건"과 "직후 사건"을 붙여야 한다.

### 6.3 3단계: 오답 선지 분류 훈련

문제를 풀고 나서 정답만 체크하면 실력이 잘 늘지 않는다. 오답 선지를 아래 5개로 분류한다.

| 오답 유형 | 예시 |
|---|---|
| 시대가 다름 | 회사령을 1920년대 사건처럼 배치 |
| 주체가 다름 | 발해 주자감을 신라 제도처럼 배치 |
| 왕이 다름 | 과거제를 성종 업적으로 배치 |
| 지역이 다름 | 만주/연해주/미주 독립운동 단체 혼동 |
| 사건 순서가 다름 | 갑신정변 결과를 동학 농민 운동 전후에 잘못 배치 |

한능검 고득점은 오답을 지우는 속도에서 나온다.

### 6.4 4단계: 시대별 우선순위

시간이 부족하면 다음 순서로 공부한다.

1. 근현대 연표: 1876~1945, 1945 이후 정부별 정책
2. 삼국·통일신라·발해 왕별 업적과 문화재
3. 고려 왕별 제도와 대외 항쟁
4. 조선 왕별 제도, 붕당, 탕평, 실학, 세도
5. 선사/초기 국가는 단서 카드로 빠르게 반복

## 7. 사이트/자료에 반영하면 좋은 형태

현재 학습 페이지는 단순 설명보다 아래 구조를 강화하는 것이 좋다.

1. 각 개념에 "출제 단서" 박스 추가
   - 예: 법화원 -> 장보고, 청해진, 산둥 반도

2. 각 개념에 "오답 함정" 박스 추가
   - 예: 독서삼품과는 원성왕, 과거제는 고려 광종

3. 통합 연표에 "사이 시기 훈련" 모드 추가
   - 예: 백제 멸망(660)과 고구려 멸망(668) 사이에 무엇이 들어가는가

4. 왕업적 페이지에 "비교 카드" 추가
   - 소수림왕 vs 법흥왕
   - 영조 vs 정조
   - 광종 vs 성종

5. 근현대는 10년 단위 필터보다 "정책 체제별" 필터가 더 유용
   - 1910년대 무단 통치
   - 1920년대 문화 통치
   - 1930~40년대 민족 말살/전시 동원

## 9. 한 줄 전략

한능검 심화는 "사건을 많이 안다"보다 "단서가 보이면 시대와 주체를 즉시 좁히고, 연표 앞뒤로 오답을 제거한다"가 더 중요하다. 따라서 공부 자료는 설명형보다 단서형, 비교형, 연표형으로 만들어야 한다.
