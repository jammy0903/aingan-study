# 한능검에서 틀리기 쉬운 부분 정리

분석 범위: 한국사능력검정시험 심화 59~75회 문항별 딥 분석 850문항

기준 자료:

- `reports/hankuksa_question_deep_analysis_59_75.json`
- `reports/hankuksa_exam_trend_analysis_59_75.md`
- `reports/hankuksa_question_deep_analysis/*.md`

이 문서는 "많이 나온 것"보다 "아는 내용인데 틀리는 지점"을 따로 분류한 것이다. 한능검 심화에서 실수는 보통 지식 부족보다 다음 세 가지에서 나온다.

1. 단서를 봤지만 시대·국가·왕을 한 칸 잘못 잡는다.
2. 사건은 알지만 앞뒤 순서를 잘못 둔다.
3. 비슷한 단체·정책·문화재를 같은 묶음으로 외워 구분하지 못한다.

## 1. 결론

한능검에서 가장 위험한 오답 구조는 "한 단어 매칭"이다.

지문에 익숙한 단어가 보이면 바로 답을 고르고 싶지만, 실제 문항은 같은 단어 주변에 다른 시대·다른 국가·다른 왕의 선지를 섞는다. 따라서 풀이 순서는 항상 아래처럼 가야 한다.

1. 단서어를 표시한다.
2. 시대/국가/왕/단체 중 무엇을 묻는지 고정한다.
3. 선지를 시대 순서와 주체 기준으로 먼저 지운다.
4. 마지막에 세부 내용이 맞는지 확인한다.

문항별 `trap` 필드 기준으로 보면 오답이 만들어지는 방식은 다음과 같다.

| 주된 함정 | 문항 수 | 비율 | 의미 |
|---|---:|---:|---|
| 보기 키워드를 시대별로 갈라야 하는 문제 | 359 | 42.2% | 단서 하나로 답을 고르면 틀리기 쉽다. 먼저 선지를 시대별로 분리해야 한다. |
| 한두 단계 앞뒤 사건을 섞는 연표 문제 | 271 | 31.9% | 사건명은 알아도 직전·직후 위치를 모르면 틀린다. |
| 국가/지역/문화권 바꿔치기 | 87 | 10.2% | 고대 국가, 발해, 가야, 독립운동 지역에서 많이 나온다. |
| 앞뒤 왕 업적 섞기 | 53 | 6.2% | 같은 나라 안에서 바로 앞뒤 왕의 제도·정책을 바꿔 낸다. |
| 원인과 결과 또는 단체 섞기 | 45 | 5.3% | 근대·일제·현대 운동사에서 많이 나온다. |
| 사진·문화재 판독 함정 | 22 | 2.6% | 이름보다 양식, 출토지, 제작 국가를 봐야 한다. |
| 정책 시행자·연도 바꾸기 | 7 | 0.8% | 조약·개혁·정부 정책을 시행 주체와 결과까지 묶어야 한다. |
| 사료 단서 과잉 매칭 | 6 | 0.7% | 단어 하나보다 시대 배경까지 함께 봐야 한다. |

## 2. 가장 틀리기 쉬운 10개 영역

### 2.1 연표 사이 시기

가장 많이 틀리는 유형은 "(가), (나) 사이" 또는 "이후의 사실"이다. 분석 데이터에서 `연표/시기 배열`은 271문항이다.

자주 걸리는 구간:

| 구간 | 틀리는 방식 | 안전한 기준 |
|---|---|---|
| 삼국 통일 | 백제 멸망, 백제 부흥운동, 백강 전투, 고구려 멸망, 나당 전쟁을 섞음 | 백제 멸망 660 -> 백제 부흥운동 -> 백강 전투 663 -> 고구려 멸망 668 -> 매소성/기벌포 |
| 고려 전기 | 태조·광종·성종 업적을 한 묶음으로 외워 섞음 | 태조 호족 포섭 -> 광종 노비안검법/과거제 -> 성종 12목/시무 28조 |
| 고려 중기~말 | 묘청, 무신정변, 몽골 침입, 원 간섭, 공민왕 개혁 순서 혼동 | 묘청 -> 무신정변 -> 최씨 정권 -> 몽골 침입 -> 원 간섭 -> 공민왕 |
| 조선 전기 | 태종·세종·세조·성종 제도 혼동 | 태종 6조 직계제 -> 세종 집현전/훈민정음 -> 세조 직전법 -> 성종 경국대전 |
| 근대 | 강화도 조약 이후 흐름을 개별 사건으로 외움 | 강화도 조약 -> 통리기무아문 -> 임오군란 -> 갑신정변 -> 동학 -> 갑오개혁 |
| 현대 | 민주화 운동과 정권 순서를 섞음 | 4.19 -> 5.16 -> 유신 -> 부마/10.26 -> 5.18 -> 6월 민주항쟁 -> 6.29 |

연표 문제는 연도 암기보다 "기준 사건의 앞/뒤"를 먼저 잡는 것이 더 중요하다.

### 2.2 선사·초기 국가 단서 착각

선사와 초기 국가는 문항 수는 많지 않지만, 틀릴 때는 대부분 단서 혼동이다.

| 헷갈리는 단서 | 답 | 같이 지울 오답 |
|---|---|---|
| 주먹도끼, 찍개, 동굴, 막집 | 구석기 | 빗살무늬 토기, 가락바퀴 |
| 빗살무늬 토기, 가락바퀴, 움집 | 신석기 | 고인돌, 비파형 동검 |
| 고인돌, 비파형 동검, 민무늬 토기, 반달 돌칼 | 청동기 | 철제 농기구, 세형동검 |
| 철제 농기구, 명도전, 붓 | 철기 | 청동기 지배층 무덤 |
| 사출도, 영고 | 부여 | 삼한의 소도, 동예의 무천 |
| 민며느리제, 가족 공동 무덤 | 옥저 | 동예 책화 |
| 책화, 무천 | 동예 | 부여 영고 |
| 천군, 소도, 제정 분리 | 삼한 | 부여 제가회의 |

오답 방지 기준: "도구-토기-무덤-경제-사회" 다섯 칸 중 두 칸 이상이 같은 시대인지 확인한다.

### 2.3 고대 국가와 왕 바꿔치기

고대는 `국가/지역 식별`과 `왕별 업적 식별`이 같이 걸린다. 특히 고구려·백제·신라가 같은 시기 주변 사건으로 묶여 나오면 한 칸씩 밀려 틀리기 쉽다.

| 비교 | 정리 |
|---|---|
| 고구려 소수림왕 vs 신라 법흥왕 | 둘 다 율령·불교 축이다. 고구려는 태학까지 묶어 소수림왕, 신라는 이차돈·건원·금관가야 병합과 묶어 법흥왕. |
| 백제 성왕 vs 신라 진흥왕 | 성왕은 사비 천도·남부여·관산성 전사, 진흥왕은 한강 장악·순수비·대가야 병합. |
| 금관가야 vs 대가야 | 금관가야는 법흥왕 때 병합, 대가야는 진흥왕 때 병합. |
| 김흠돌의 난 vs 김헌창의 난 | 김흠돌의 난은 신문왕 왕권 강화, 김헌창의 난은 신라 하대 혼란. |
| 독서삼품과 vs 과거제 | 독서삼품과는 신라 원성왕, 과거제는 고려 광종. |
| 장보고 청해진 vs 장보고의 난 | 청해진은 해상 무역 거점, 장보고의 난은 신라 하대 중앙 귀족과 충돌. |

고대사는 나라 이름만 보는 순간 위험하다. 수도, 문화재, 전쟁 상대, 병합 주체까지 같이 확인해야 한다.

### 2.4 발해와 통일신라 혼동

발해는 신라와 같은 남북국 시대로 묶여 나오기 때문에 제도·문화 단서가 자주 섞인다.

| 단서 | 답 |
|---|---|
| 주자감, 3성 6부, 정당성, 선조성, 중대성 | 발해 |
| 인안 | 발해 무왕 |
| 대흥, 수도 상경 천도, 신라도 | 발해 문왕 |
| 해동성국 | 발해 선왕 |
| 9주 5소경, 국학, 관료전, 녹읍 폐지 | 통일신라 신문왕 |
| 독서삼품과 | 통일신라 원성왕 |

오답 방지 기준: 발해는 당 제도 변형과 고구려 계승 단서, 통일신라는 골품제·9주 5소경·국학 단서로 분리한다.

### 2.5 고려 왕별 업적과 대외 항쟁

고려는 분석 데이터에서 가장 많은 시대 표지가 잡혔고, 연표형도 56문항으로 가장 많다. 왕별 제도와 대외 항쟁이 같이 꼬인다.

| 헷갈리는 축 | 반드시 분리할 것 |
|---|---|
| 태조 vs 광종 vs 성종 | 태조=훈요 10조·호족 포섭, 광종=노비안검법·과거제, 성종=12목·최승로 시무 28조 |
| 서희 vs 강감찬 | 서희=외교 담판·강동 6주, 강감찬=귀주대첩 |
| 윤관 vs 별무반 | 숙종 때 별무반 편성, 예종 때 윤관의 동북 9성 |
| 묘청 vs 무신정변 | 묘청의 서경 천도 운동 이후 무신정변 |
| 최충헌 vs 최우 | 최충헌=교정도감, 최우=정방·삼별초 |
| 몽골 침입 vs 원 간섭 | 강화도 천도·삼별초는 몽골 항쟁, 쌍성총관부·정동행성은 원 간섭 |
| 공민왕 개혁 | 쌍성총관부 수복, 전민변정도감, 신돈, 반원 개혁 |

고려는 "왕 이름 -> 제도"보다 "왕 이름 -> 바로 앞뒤 왕 -> 대외 관계"로 외워야 한다.

### 2.6 조선 왕·붕당·제도 섞기

조선은 전기 왕 업적과 후기 붕당·탕평·세도가 서로 다른 축인데, 시험에서는 같은 선지 안에서 섞인다.

| 비교 | 구분 |
|---|---|
| 태종 vs 세종 | 태종은 사병 혁파·6조 직계제, 세종은 집현전·훈민정음·4군 6진. |
| 세조 vs 성종 | 세조는 계유정난·직전법, 성종은 경국대전·홍문관·관수관급제. |
| 연산군 vs 중종 | 연산군은 무오·갑자사화, 중종은 조광조·현량과·기묘사화. |
| 광해군 vs 인조 | 광해군은 중립 외교·대동법, 인조는 인조반정·정묘호란·병자호란. |
| 숙종 vs 영조 vs 정조 | 숙종=환국, 영조=탕평·균역법, 정조=규장각·장용영·수원 화성. |
| 정조 vs 순조 | 정조는 개혁 군주, 순조는 세도 정치 시작·신유박해·홍경래의 난. |

오답 방지 기준: 조선 문제는 왕만 보지 말고 "제도/전쟁/붕당/경제/문화" 중 어떤 축인지 먼저 표시한다.

### 2.7 근대 조약·개혁 순서

근대는 개별 사건 설명을 알아도 순서를 놓치면 틀린다. 특히 임오군란, 갑신정변, 동학 농민 운동, 갑오개혁이 서로 결과처럼 섞인다.

핵심 흐름:

흥선대원군 통상 수교 거부 -> 강화도 조약 -> 개화 정책/통리기무아문 -> 임오군란 -> 갑신정변 -> 동학 농민 운동 -> 갑오개혁 -> 을미사변 -> 아관파천 -> 독립협회 -> 대한제국/광무개혁 -> 을사늑약 -> 정미7조약/군대 해산 -> 국권 피탈

| 함정 | 정리 |
|---|---|
| 통리기무아문 위치 | 강화도 조약 이후, 임오군란 이전. |
| 갑신정변 결과 | 청의 개입, 한성 조약, 톈진 조약으로 연결. |
| 동학 농민 운동 결과 | 전주 화약, 집강소, 청일 전쟁, 갑오개혁으로 연결. |
| 독립협회 | 만민공동회, 관민공동회, 중추원 개편 요구. |
| 광무개혁 | 대한제국, 황제권 강화, 양전 사업, 지계 발급. |
| 의병 vs 애국계몽 | 의병은 무장 투쟁, 애국계몽은 교육·산업·언론 중심. |

근대는 사건별 설명 카드보다 "원인 -> 사건 -> 조약/개혁/결과" 카드가 더 안전하다.

### 2.8 일제 강점기 연도대와 단체

일제 강점기는 1910년대, 1920년대, 1930~40년대만 정확히 갈라도 오답이 크게 줄어든다.

| 시기 | 핵심 단서 | 자주 섞이는 오답 |
|---|---|---|
| 1910년대 | 무단 통치, 헌병 경찰, 조선 태형령, 토지 조사 사업, 회사령 | 산미 증식 계획, 신간회 |
| 1920년대 | 문화 통치, 산미 증식 계획, 물산 장려 운동, 민립 대학 설립 운동, 신간회 | 국가 총동원법, 창씨개명 |
| 1930~40년대 | 민족 말살 통치, 국가 총동원법, 황국 신민 서사, 창씨개명, 징병제 | 헌병 경찰, 회사령 |

독립운동 단체도 지역과 활동으로 나눠야 한다.

| 단체/운동 | 구분 단서 |
|---|---|
| 의열단 | 김원봉, 조선 혁명 선언, 개인 의거·폭력 투쟁 노선 |
| 한인 애국단 | 김구, 이봉창, 윤봉길 |
| 신민회 | 105인 사건, 대성 학교, 오산 학교, 신흥 강습소 |
| 신간회 | 정우회 선언, 민족 유일당, 광주 학생 항일 운동 지원 |
| 한국 독립군 | 지청천, 쌍성보·대전자령 |
| 조선 혁명군 | 양세봉, 영릉가·흥경성 |
| 한국광복군 | 대한민국 임시 정부, 지청천, 국내 진공 작전 준비 |

오답 방지 기준: 단체명만 보지 말고 "인물-지역-전투/활동" 3칸을 채운다.

### 2.9 현대 민주화 운동과 정부별 정책

현대사는 민주화 운동 순서와 정부별 정책이 분리되어야 한다.

민주화 흐름:

### 2.10 문화재·지역 사진형

문화재 문제는 이름이 아니라 "양식, 위치, 제작 국가"를 묻는다. 분석 데이터에서는 `문화재/자료 판독`이 22문항이지만, 실제 체감 난도는 높다.

## 3. 시대별 시험 직전 체크리스트

### 선사·초기 국가

- 구석기/신석기/청동기/철기는 도구 하나로만 판단하지 않는다.
- 고인돌은 청동기, 세형동검·잔무늬거울은 철기 계열로 분리한다.
- 부여·옥저·동예·삼한은 제천 행사, 혼인 풍습, 읍락 질서, 제정 분리로 구분한다.

### 삼국·남북국

- 율령·불교 단서가 나오면 어느 나라 왕인지 먼저 본다.
- 성왕과 진흥왕은 관산성·한강·대가야 병합으로 분리한다.
- 통일신라 중대와 하대는 신문왕 왕권 강화와 김헌창·장보고·원종/애노로 나눈다.
- 발해는 무왕·문왕·선왕, 주자감, 3성 6부를 한 묶음으로 본다.

### 고려

- 태조·광종·성종은 고려 전기 왕권 강화 3칸으로 고정한다.
- 거란은 서희와 강감찬을 구분한다.
- 무신정변, 최씨 정권, 몽골 침입, 원 간섭, 공민왕 개혁은 순서로 외운다.
- 문화재는 팔만대장경, 직지, 상감 청자, 불교 건축 양식을 분리한다.

### 조선

- 태종·세종·세조·성종은 제도별로 비교한다.
- 사화는 연산군/중종, 붕당은 선조 이후, 환국은 숙종으로 잡는다.
- 영조와 정조는 탕평이라는 공통점 때문에 섞이지만, 균역법은 영조, 규장각·장용영은 정조다.
- 순조는 세도 정치·신유박해·홍경래의 난으로 끝을 잡는다.

### 근대

- 강화도 조약 이후 흐름은 반드시 통째로 암기한다.
- 임오군란과 갑신정변은 원인·개입 세력·결과 조약이 다르다.
- 동학 농민 운동은 전주 화약, 집강소, 청일 전쟁, 갑오개혁으로 연결한다.
- 을사늑약, 헤이그 특사, 정미7조약, 군대 해산, 의병 확대를 한 줄로 본다.

### 일제 강점기

- 1910/1920/1930~40년대 정책을 먼저 나눈다.
- 신간회는 정우회 선언, 광주 학생 항일 운동 지원과 연결한다.
- 무장 독립운동은 단체명보다 지역·전투·인물로 구분한다.
- 사회·문화 운동은 물산 장려, 민립 대학, 브나로드, 형평 운동을 섞지 않는다.

### 현대

- 광복 직후는 모스크바 3국 외상 회의, 좌우 합작, 남북 협상, 정부 수립 순서를 확인한다.
- 4.19와 5.16을 뒤집지 않는다.
- 5.18은 전두환 집권 과정, 6월 민주항쟁은 대통령 직선제 개헌과 연결한다.
- 통일 정책은 정부별로 분리한다.

## 4. 공부 자료에 반영할 형식

개념 페이지나 퀴즈 카드에는 다음 네 칸을 붙이는 것이 좋다.

| 카드 칸 | 예시 |
|---|---|
| 결정 단서 | `정우회 선언` |
| 바로 연결 | 신간회, 민족 유일당 운동, 1927 |
| 헷갈리는 오답 | 의열단, 한인 애국단, 조선 혁명군 |
| 제거 기준 | 단체 성격이 합법 대중운동인지, 의거/무장투쟁인지 먼저 분리 |

특히 다음 유형은 별도 훈련 모드로 만들 가치가 있다.

1. 사이 시기 훈련: 사건 A와 사건 B 사이에 들어갈 사건 고르기.
2. 왕 업적 비교: 서로 붙어 있는 왕 2~3명을 한 화면에서 비교.
3. 단체 식별: 인물·지역·활동을 보고 단체명 맞히기.
4. 문화재 판독: 사진 없이도 양식·지역·제작 국가 단서로 맞히기.
5. 오답 선지 분류: 시대 오류, 주체 오류, 순서 오류, 지역 오류로 표시.
